03
Pipeline AI Agent chi tiết
Từng bước từ idea đến video hoàn chỉnh
Mục tiêu: Mỗi ngày hệ thống tự động tạo 3–5 video ngắn (TikTok/Reels/Shorts) liên quan đến chủ đề kênh, kèm affiliate link, đăng theo lịch giờ vàng — không cần can thiệp thủ công hàng ngày.

Bước 1 — Tìm trending topics & chọn sản phẩm affiliate

Agent đầu tiên chạy mỗi sáng 6:00 AM tự động:

  • Gọi Google Trends API lấy top trending trong ngành (ví dụ: du lịch, lifestyle, công nghệ)
  • Crawl top 10 TikTok/Reels đang viral trong niche bằng Apify hoặc Phantombuster
  • Dùng Claude API phân tích, chọn 5 chủ đề phù hợp với kênh và có affiliate product liên quan
  • Tạo content calendar cho ngày hôm đó, lưu vào Google Sheet hoặc Airtable
n8n Workflow · Content Planner Agent

Mỗi ý tưởng bao gồm: tiêu đề, hook mở đầu, sản phẩm affiliate phù hợp. Format JSON." }] } // Node 4: Lưu vào Google Sheet Google Sheets → Append rows với data từ Claude

Bước 2 — Viết kịch bản tối ưu

Prompt structure cho Script Writer Agent:

Claude System Prompt · Script Writer
SYSTEM: Bạn là chuyên gia viết kịch bản video ngắn viral cho TikTok/Reels/Shorts. Luôn viết bằng tiếng Việt. Phong cách: gần gũi, tự nhiên, như người bạn kể chuyện. Cấu trúc bắt buộc (45–60 giây đọc): 1. HOOK (3–5 giây): Câu mở gây tò mò hoặc shock 2. BRIDGE (5 giây): Kết nối hook với nội dung chính 3. VALUE (30–40 giây): Nội dung hữu ích, dễ nhớ 4. CTA (5–7 giây): Kêu gọi hành động + mention sản phẩm affiliate tự nhiên Sau kịch bản, thêm: - CAPTION: 150 ký tự + hashtags - AFFILIATE_NOTE: Cách gắn link tự nhiên vào caption USER: Chủ đề: {{topic}} Sản phẩm affiliate: {{product_name}} - {{product_link}} Nền tảng đăng: {{platform}}

Tối ưu A/B testing tự động

  • Tạo 2 phiên bản caption khác nhau bằng Claude (A: emotional, B: informational)
  • Đăng phiên bản A lúc 19:00, phiên bản B lúc 20:30
  • Sau 24h: agent đọc analytics, so sánh view/engagement
  • Lưu pattern thắng vào database để prompt lần sau

Cách gắn affiliate link không bị block

  • Linktree / bio.link: Tập hợp tất cả link tại 1 URL duy nhất, đặt trong bio. Agent tự cập nhật khi có sản phẩm mới.
  • Comment ghim: YouTube + Facebook cho phép ghim comment có link. Agent tự post comment sau khi video up.
  • Caption tự nhiên: Claude viết câu CTA không spam: "Mình mua ở link dưới comment nè" thay vì "CLICK NOW".
  • URL shortener riêng: Dùng yourls.org self-host để track click riêng, tránh bị filter.
  • Story/Swipe-up: Facebook/Instagram Stories với link trực tiếp khi đủ follower.

GIAI ĐOẠN 1 · Ngày 1–30
Xây nền tảng
  • Chọn 1 niche duy nhất — đừng làm tất cả (ví dụ: "Du lịch Phan Thiết giá rẻ")
  • Tạo avatar AI nhất quán (tên, giọng nói, phong cách visual)
  • Setup pipeline n8n + test toàn bộ flow với 5 video đầu thủ công
  • Đăng đều đặn 1 video/ngày, không bỏ ngày nào trong tháng đầu
  • Mục tiêu: 500 followers TikTok (điều kiện bật TikTok Shop/link)
GIAI ĐOẠN 2 · Ngày 31–90
Scale nội dung
  • Tăng lên 2–3 video/ngày bằng cách chạy pipeline song song
  • Thêm kênh thứ 2 (Facebook Page) cùng nội dung, chỉ re-purpose
  • Bật affiliate cho top 3 sản phẩm liên quan niche
  • A/B test format: hook dạng câu hỏi vs câu statement
  • Mục tiêu: 1 video viral/tuần (>10K views), 2,000 followers
GIAI ĐOẠN 3 · Tháng 4–6
Monetize & mở rộng
  • Đăng ký TikTok Creator Fund / YouTube Partner Program
  • Mở thêm 2–3 kênh niche phụ (agent tự tái sử dụng pipeline)
  • Thêm sản phẩm digital affiliate (khóa học, ebook) — hoa hồng cao hơn
  • Xây email list từ traffic video — dài hạn nhất
  • Mục tiêu: $200–500/tháng từ affiliate + ads

import anthropic, requests, json from datetime import datetime, timedelta # ─── Config ─────────────────────────────────────────── CLAUDE_KEY = "sk-ant-..." ELEVENLABS_KEY = "..." HEYGEN_KEY = "..." CREATOMATE_KEY = "..." BUFFER_TOKEN = "..." VOICE_ID = "vietnamese_voice_id" AVATAR_ID = "your_heygen_avatar_id" AFFILIATE_LINK = "https://accesstrade.vn/go/..." # ─── Step 1: Viết kịch bản ──────────────────────────── def generate_script(topic: str, product: str) -> dict: client = anthropic.Anthropic(api_key=CLAUDE_KEY) response = client.messages.create( model="claude-sonnet-4-20250514", max_tokens=1000, messages=[{ "role": "user", "content": f"""Viết kịch bản video TikTok 45 giây về: {topic} Sản phẩm mention: {product} Trả về JSON: {{script, caption, hashtags, cta}}" }] ) return json.loads(response.content[0].text) # ─── Step 2: Text-to-Speech ─────────────────────────── def text_to_speech(text: str) -> str: url = f"https://api.elevenlabs.io/v1/text-to-speech/{VOICE_ID}" headers = {"xi-api-key": ELEVENLABS_KEY, "Content-Type": "application/json"} res = requests.post(url, headers=headers, json={"text": text, "model_id": "eleven_multilingual_v2"}) # Upload audio lên S3 / Google Drive, return URL return upload_to_storage(res.content, "audio.mp3") # ─── Step 3: Tạo Avatar Video (HeyGen) ─────────────── def create_avatar_video(audio_url: str) -> str: res = requests.post("https://api.heygen.com/v2/video/generate", headers={"X-Api-Key": HEYGEN_KEY}, json={ "video_inputs": [{ "character": {"type": "avatar", "avatar_id": AVATAR_ID}, "voice": {"type": "audio", "audio_url": audio_url} }], "dimension": {"width": 720, "height": 1280} }) video_id = res.json()["data"]["video_id"] return poll_heygen(video_id) # đợi render xong, return URL # ─── Step 4: Render final video (Creatomate) ───────── def render_video(avatar_url: str, caption: str) -> str: res = requests.post("https://api.creatomate.com/v1/renders", headers={"Authorization": f"Bearer {CREATOMATE_KEY}"}, json={ "template_id": "your-template-id", "modifications": { "avatar_clip": avatar_url, "subtitle_text": caption[:50] } }) render_id = res.json()["id"] return poll_creatomate(render_id) # ─── Step 5: Schedule đăng (Buffer) ────────────────── def schedule_post(video_url: str, caption: str, platform_id: str): post_time = get_golden_hour() # tính giờ vàng ngày hôm sau full_caption = caption + f"\n\n🔗 Link: {AFFILIATE_LINK}" requests.post("https://api.bufferapp.com/1/updates/create.json", headers={"Authorization": f"Bearer {BUFFER_TOKEN}"}, json={ "profile_ids": [platform_id], "text": full_caption, "media": {"video": video_url}, "scheduled_at": post_time.isoformat() }) # ─── Main orchestrator ──────────────────────────────── def run_pipeline(topic: str, product: str): print(f"🚀 Bắt đầu tạo video: {topic}") data = generate_script(topic, product) audio_url = text_to_speech(data["script"]) avatar_url = create_avatar_video(audio_url) video_url = render_video(avatar_url, data["caption"]) schedule_post(video_url, data["caption"], "tiktok_profile_id") schedule_post(video_url, data["caption"], "facebook_profile_id") print("✅ Video đã được lên lịch đăng!") # Chạy run_pipeline("Top 5 quán ăn ngon ở Phan Thiết", "Tour Mũi Né Klook")
